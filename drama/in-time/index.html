【又名】我可能…不會愛你 / In Time With You / イタズラな戀愛白書
【導演】瞿友寧
【主演】林依晨 / 陳柏霖 / 王陽明 / 陳匡怡
【類型】偶像 / 愛情 / 都市
【首播】2011年09月18日
【集數】23集
【簡介】《我可能不會愛你》講述「一個關於『特別的朋友』的故事」。 程又青(林依晨 飾)，百貨公司的鞋區區長，交過幾個男朋友，傷心、傷身又傷情。 三十歲那天，她還是孤家寡人，更被提醒初老癥狀的來臨，只好和唯一真心祝福生日快樂的他打賭…。 李大仁(陳柏霖 飾)，是航空公司的地勤督導，年輕有為，交過幾任女友，但戀情都很短暫！ 兩個從高中開始就不對盤的男女，卻陰錯陽差的一直在一起，卻又一直是一對談不成戀愛的朋友。於是兩人打了賭… 「三十五歲前誰先結婚，紅包---十萬！！」的賭注，讓驕傲的她開始用力搜尋周圍可能結婚的對象。誰說女人過了30就是拉警報即將滯銷的即期品？她就會是那個例外---一個充滿魅力的熟女！ 他曾對她說過，我可能不會愛你。換來了她的一句:謝謝，感激不盡。 他不知道為什麼，交女友的條件只有一個，必須得忍受"她"的存在；也不知道為什麼，怎麼樣都放不下她一個人；更不知道為什麼自己和她的世界全都卡在一塊…是因為太像家人了，所以不可能在一起？ 他曾經寫了一首歌給她，卻只能孤零零的唱著。很久很久以後，她才知道，歌詞中的我可能不會愛你，其實是………我愛你。
