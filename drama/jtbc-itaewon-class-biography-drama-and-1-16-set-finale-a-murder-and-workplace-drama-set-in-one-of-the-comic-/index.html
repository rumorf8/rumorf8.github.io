隨後他在梨泰院開了一家叫「蜜夜」的小酒館，並向害死他爸爸的仇人報仇的故事。

《梨泰院CLASS》改編自人氣網路漫畫，將由曾執導過《雲畫的月光》金成允導演執導，

朴敘俊接到了男主角的出演提議。

人物簡介

朴敘俊飾樸賽路

這是一部集命案和職場劇於一體的漫畫，男主角憑藉著自己的努力開了一家餐廳，為了向害死他父親的人復仇。

《梨泰院Class》改編自同名人氣網漫，講述土湯匙出身的樸賽路爸爸死後陷入了人生的低谷，隨後他在梨泰院開了一家叫「蜜夜」的小酒館，並向害死他爸爸的仇人報仇。

他是不會對不正義妥協的直進型青年，因為無法抹去的風怒他投身到梨泰院，

開始挑戰全新的夢想，他將會對餐飲界大企業「張家」展開痛快的反擊。

權娜拉飾吳秀雅

權娜拉有望加盟JTBC新劇《 #梨泰院Class#》，

將在劇中飾演吳秀雅一角。


金多美飾演趙以瑞

她是有著神賜予的頭腦的「高智能」且具備獨特魅力的人物，

是SNS明星也是網絡影響力人物，具備天使般的外貌和反轉性格。

她和朴塞路因為惡緣而結緣並一起在梨泰院發展。

劉在明飾演餐飲界大企業「張家」的會長張大熙

他因為小時候餓肚子的記憶所以投身餐飲業，憑藉一己之力成為以他為中心的財閥，

是一個權威主義者。因為兒子的意外，

他再次和眼中釘樸塞路爭鋒相對，如同鐵壁的他的人生逐漸動搖。

金東希飾演張大熙（ 劉在明   飾）會長的二兒子張根秀

因為是庶子，他被同父異母的哥哥張根元虐待時父母也只是袖手旁觀。

17歲獨立的張根秀單戀朋友趙以瑞（ 金多美飾），

同時通過樸賽路（ 樸敘俊飾）認識並理解了什麼才是"真正的大人”。

《梨泰院Class》改編自同名網漫，講述在不合理的世上，

因為意志和活力聚集在一起的年輕人們的「hip」的反叛故事。

在微縮世界梨泰院這個小街道上，

自由追求著各自價值觀，譜寫創業神話。
由《雲畫的月光》金成允導演執導，漫畫作者趙光鎮親自執筆，

預計接檔《巧克力》播出。
