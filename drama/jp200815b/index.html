【又名】在著名建築享用午餐 / 名建築で晝食を
【編劇】橫幕智裕
【導演】吉見拓真
【主演】池田依來沙 / 田口智朗 / 小川紗良 / 三上寬
【類型】日劇 / 劇情
【首播】2020年08月15日
【集數】10集
【簡介】《在名建築里吃午餐》是以建築和美食為主題的新類型劇集。故事講述夢想開咖啡店的春野藤（池田依來沙）是個公司職員，她在社交網路認識了中年建築模型士植草千明（田口智朗），他喜歡「少女建築」，會走訪不同的名建築物，本身在工作及戀愛煩惱的春野，在和他一起四處走的時候，會因為人生經驗豐富的他的一言而醒覺，得到勇氣向前走。 劇集將會拍攝日本多座著名建築物，例如日本現存最古老的啤酒館「BEER HALL LION銀座七丁目店」，有機建築大師法蘭克·洛伊·萊特設計的「自由學園明日館」，以文化人酒店見稱的「山上酒店」，白洲次郎夫妻的舊邸宅「武相庄」等，兩位主角會在這些建築物中享用美味的午餐。拍攝以4K高清進行。
